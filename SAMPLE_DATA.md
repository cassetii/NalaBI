## Cara Input ke Dashboard:

1. Login ke dashboard
2. Klik tab "Tambah Project"
3. Copy paste data dari sample di atas
4. Pastikan format tanggal: YYYY-MM-DD (contoh: 2025-11-15)
5. Pastikan nilai kontrak tanpa titik/koma (contoh: 450000000)
6. Klik "Simpan Project"
7. Ulangi untuk semua sample

## Tips:

- Koordinat di atas adalah lokasi real di Makassar
- Semua data bisa diedit setelah disimpan
- Status bisa diupdate sesuai progress real
- Gunakan sample ini untuk demo ke client/team
